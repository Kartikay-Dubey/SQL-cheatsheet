SQL Commands and Functions

Command/Function Description Example
SQL Commands - DDL (Data Definition Language)
CREATE Creates a new database object like a table, database, schema, or index. Defines structure and constraints. CREATE TABLE students (id INT PRIMARY KEY, name VARCHAR(50));
ALTER Modifies an existing database object, such as adding columns, changing data types, or adding constraints. ALTER TABLE students ADD email VARCHAR(100);
DROP Permanently deletes a database object, including all its data and structure. DROP TABLE students;
TRUNCATE Removes all rows from a table but retains its structure for future use. Faster than DELETE for clearing data. TRUNCATE TABLE students;
RENAME Changes the name of a database object, such as a table or column. RENAME TABLE students TO pupils;
SQL Commands - DML (Data Manipulation Language)
INSERT Adds new rows of data into a table. Can insert single or multiple rows. INSERT INTO students (id, name) VALUES (1, 'John');
UPDATE Modifies existing data in a table based on a condition. UPDATE students SET name = 'Jane' WHERE id = 1;
DELETE Removes rows from a table based on a specified condition. DELETE FROM students WHERE id = 1;
SQL Commands - DQL (Data Query Language)
SELECT Retrieves data from one or more tables. Supports filtering, sorting, grouping, and joins. SELECT * FROM Students;
SELECT DISTINCT Retrieves unique values from a column, eliminating duplicates. SELECT DISTINCT name FROM students;
SELECT with ORDER BY Sorts query results in ascending (ASC) or descending (DESC) order. SELECT name FROM students ORDER BY name ASC;
SELECT with GROUP BY Groups rows with similar values for use with aggregate functions. SELECT dept, COUNT(*) FROM students GROUP BY dept;
SELECT with HAVING Filters grouped results based on a condition (used with GROUP BY). SELECT dept, COUNT(*) FROM students GROUP BY dept HAVING COUNT(*) > 2;
SQL Commands - DCL (Data Control Language)
GRANT Assigns specific privileges (e.g., SELECT, INSERT) to a user or role. GRANT SELECT, INSERT ON students TO user1;
REVOKE Removes previously granted privileges from a user or role. REVOKE SELECT ON students FROM user1;
SQL Commands - TCL (Transaction Control Language)
COMMIT Permanently saves all changes made during the current transaction. COMMIT;
ROLLBACK Reverts all changes made during the current transaction to the last commit or savepoint. ROLLBACK;
SAVEPOINT Sets a point within a transaction to which you can later rollback. SAVEPOINT savepoint1;
SET TRANSACTION Defines properties of a transaction, such as read-only or read-write mode. SET TRANSACTION READ ONLY;
SQL Commands - Utility Commands
USE Selects a specific database to work with in the current session. USE school_db;
DESCRIBE/DESC Displays the structure of a table, including column names, data types, and constraints. DESCRIBE students;
SHOW Lists database objects, such as tables, databases, or indexes. SHOW TABLES;
EXPLAIN Provides details about how a query will be executed, including the query plan. EXPLAIN SELECT * FROM students;
SQL Commands - Index Management
CREATE INDEX Creates an index on one or more columns to improve query performance. CREATE INDEX idx_name ON students(name);
DROP INDEX Removes an index from a table, reducing overhead but possibly slowing queries. DROP INDEX idx_name ON students;
SQL Commands - Joins and Set Operations
INNER JOIN Combines rows from two tables where the join condition is met. SELECT s.name, d.dept_name FROM students s INNER JOIN departments d ON s.dept_id = d.id;
LEFT JOIN Returns all rows from the left table and matching rows from the right table (NULL for non-matches). SELECT s.name, d.dept_name FROM students s LEFT JOIN departments d ON s.dept_id = d.id;
RIGHT JOIN Returns all rows from the right table and matching rows from the left table (NULL for non-matches). SELECT s.name, d.dept_name FROM students s RIGHT JOIN departments d ON s.dept_id = d.id;
FULL JOIN Returns all rows from both tables, with NULLs where thereâ€™s no match. SELECT s.name, d.dept_name FROM students s FULL JOIN departments d ON s.dept_id = d.id;
UNION Combines results of two SELECT queries, removing duplicates. SELECT name FROM students UNION SELECT name FROM teachers;
UNION ALL Combines results of two SELECT queries, including duplicates. SELECT name FROM students UNION ALL SELECT name FROM teachers;
Inbuilt Functions - String Functions
CONCAT Combines two or more strings into a single string. SELECT CONCAT(name, ' ', email) AS full_info FROM students;
UPPER Converts a string to uppercase. SELECT UPPER(name) FROM students;
LOWER Converts a string to lowercase. SELECT LOWER(name) FROM students;
TRIM Removes leading and trailing spaces (or specified characters) from a string. SELECT TRIM(' hello ');
LTRIM Removes leading spaces (or specified characters) from a string. SELECT LTRIM(' hello');
RTRIM Removes trailing spaces (or specified characters) from a string. SELECT RTRIM('hello ');
SUBSTRING/SUBSTR Extracts a portion of a string based on start position and length. SELECT SUBSTRING(name, 1, 3) FROM students;
REPLACE Replaces all occurrences of a substring with another string. SELECT REPLACE(name, 'John', 'Jon') FROM students;
Inbuilt Functions - Numeric Functions
ABS Returns the absolute (positive) value of a number. SELECT ABS(-10);
ROUND Rounds a number to a specified number of decimal places. SELECT ROUND(123.456, 2);
CEIL/CEILING Returns the smallest integer greater than or equal to a number. SELECT CEIL(123.1);
FLOOR Returns the largest integer less than or equal to a number. SELECT FLOOR(123.9);
MOD Returns the remainder of a division operation. SELECT MOD(10, 3);
POWER Raises a number to a specified power. SELECT POWER(2, 3);
Inbuilt Functions - Date/Time Functions
NOW Returns the current date and time. SELECT NOW();
CURDATE/CURRENT_DATE Returns the current date. SELECT CURDATE();
CURTIME/CURRENT_TIME Returns the current time. SELECT CURTIME();
DATEADD Adds a specified time interval to a date. SELECT DATEADD(day, 10, '2025-04-01');
DATEDIFF Returns the difference between two dates in days (or other units). SELECT DATEDIFF('2025-04-10', '2025-04-01');
EXTRACT Extracts a part (e.g., year, month) from a date. SELECT EXTRACT(YEAR FROM '2025-04-10');
Inbuilt Functions - Conditional Functions
IF Returns a value based on a condition (true/false). SELECT IF(marks >= 50, 'Pass', 'Fail') FROM students;
IFNULL Returns a specified value if the expression is NULL, otherwise returns the expression. SELECT IFNULL(email, 'No Email') FROM students;
COALESCE Returns the first non-NULL value from a list of expressions. SELECT COALESCE(email, phone, 'No Contact') FROM students;
CASE Evaluates multiple conditions and returns a value when the first condition is met. SELECT name, CASE WHEN marks >= 50 THEN 'Pass' ELSE 'Fail' END FROM students;
Length Functions
LENGTH Returns the length of a string in bytes (may differ for multi-byte characters). SELECT LENGTH(name) FROM students;
CHAR_LENGTH/CHARACTER_LENGTH Returns the length of a string in characters (handles multi-byte characters correctly). SELECT CHAR_LENGTH(name) FROM students;
BIT_LENGTH Returns the length of a string in bits. SELECT BIT_LENGTH(name) FROM students;
Aggregate Functions
COUNT Counts the number of rows or non-NULL values in a column. SELECT COUNT(*) FROM students;
SUM Calculates the total sum of numeric values in a column. SELECT SUM(marks) FROM students;
AVG Calculates the average of numeric values in a column. SELECT AVG(marks) FROM students;
MIN Returns the smallest value in a column (works with numbers, strings, dates). SELECT MIN(marks) FROM students;
MAX Returns the largest value in a column (works with numbers, strings, dates). SELECT MAX(marks) FROM students;
